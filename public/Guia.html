Guía de 5 Pasos

Completa cada paso cuidadosamente para cargar y analizar tu archivo en Cihuata Data-Lab.

Paso 1 — Selecciona o arrastra tu archivo

En Inicio, haz clic en el área “Carga de Datos” o arrastra un archivo CSV o XLSX hacia el recuadro. Verás el nombre del archivo cuando se haya detectado.

Paso 2 — Revisa la previsualización

Comprueba el nombre y el tamaño del archivo en la tarjeta de previsualización. Si no corresponde, elimínalo con el icono ✕ y vuelve a seleccionar otro.

Paso 3 — Sube el archivo y espera la barra de progreso

Al comenzar la carga verás una barra de progreso y el porcentaje. No cierres la ventana hasta que el proceso termine al 100%.

Paso 4 — Analiza

Cuando la carga termine, el botón “Analizar” se activará. Presiónalo para iniciar el análisis de tu dataset.

Paso 5 — Abrir Power BI y actualizar

  1. Abre Power BI Desktop en tu equipo.
  2. Conecta la fuente de datos MySQL usando las credenciales del usuario.
  3. Selecciona las vistas v_empleados y vw_prp_predicciones.
  4. En la cinta superior, haz clic en Inicio → Actualizar (Refresh) para traer los datos nuevos desde la base de datos.

Paso 6 — ¿Problemas o errores?

  • Confirma que el formato sea CSV/XLSX y que el archivo no esté protegido.
  • Evita caracteres especiales en el nombre (usa letras, números y guiones bajos).
  • Si el archivo es muy grande, prueba con una muestra reducida para validar.
